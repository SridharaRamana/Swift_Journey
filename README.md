🚆 SwiftJourney – Train Ticket Exchange Platform

SwiftJourney is a web platform that allows users to share unused train tickets with other travelers who urgently need them.
Instead of tickets going to waste, users can post their travel details, and other users can contact them to obtain the ticket.

This project was built using HTML, CSS, JavaScript, PHP, and MySQL and runs locally using XAMPP.

🧠 Problem This Project Solves

Sometimes travelers book train tickets but cannot travel due to unexpected issues.

Instead of letting the ticket go unused, SwiftJourney allows them to:

Post ticket travel details

Other travelers search for the route

Contact the ticket owner

Complete the exchange privately

This helps travelers save money and time.
